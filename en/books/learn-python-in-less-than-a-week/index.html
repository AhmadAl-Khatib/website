Chapter -1- A Tour of Python

  1. 1- The print() Function

    The `print()` function is the first function we will learn in the Python language. It is used to display text on the screen, and it is the most important function used to communicate with the user and the programmer by displaying outputs to them on the screen. It is also used to debug the code and determine the value of a specific variable... The `print()` function code is written as follows:

    print("The text you want to display on the screen")

    As an example:

    print("Hello World!")

    The result will be:

    Hello World!

    And it will be displayed on the output screen.

    If we want to display a number or a variable on the screen using the `print()` function, we write the code as follows:

    print(4)

    The result will be:

    4

    My_name = "Ahmad"

    print(My_name)

    The result will be:

    Ahmad

    Note: We place the text we want to display on the screen within double quotes (" ") or single quotes (' '), while the variables and numbers are not placed within quotes.

    We can also combine a text with a variable or a number, and display it on the screen:

    My_name = "Ahmad"

    print("Hello " + My_name)

    The result will be:

    Hello Ahmad

    Note: We use the (+) operator between the text and the variable (if we want to combine them), and we place a space after the text 'Hello ' to separate the two words and not display them stuck together.

  2. 2- Variable and Value

    In programming, a variable is a symbol or name we give to a specific location in the device's memory, where we store data that we can use, modify, or delete, and it is a type of data storage container.

    The value is the data that we store in the variable, whether it is a number, a text, a list, etc.

    As an example:

    my_first_name = "Ahmad"

    In this example, the variable is my_first_name, and the value is the text "Ahmad".

    Naming rules for variables:

    • It must start with a letter or an underscore (_).
    • It cannot start with a number.
    • It can only contain letters (a-z), numbers (0-9), and underscores (_).
    • Variable names are case-sensitive (e.g., name is not the same as Name).
    • Keywords cannot be used as variable names (e.g., `print`, `for`, `if`...).
  3. 3- The title(), lower(), and upper() Methods

    These methods are used to modify the display of a text (a string). We use them as follows:

    name = "ahmad al khatib"

    print(name.title())

    The result will be:

    Ahmad Al Khatib

    The `title()` method capitalizes the first letter of each word in the string.

    print(name.lower())

    The result will be:

    ahmad al khatib

    The `lower()` method converts all letters in the string to lowercase.

    print(name.upper())

    The result will be:

    AHMAD AL KHATIB

    The `upper()` method converts all letters in the string to uppercase.
